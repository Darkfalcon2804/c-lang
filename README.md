## 🚀 Getting Started
1. **Clone the repository**  
   ```bash
   git clone https://github.com/Darkfalcon2804/c-lang.git
   ```
2. **Navigate to the project folder**  
   ```bash
   cd c-lang
   ```
3. **Compile and run a C program**  
   ```bash
   gcc filename.c -o output
   ./output
   ```
